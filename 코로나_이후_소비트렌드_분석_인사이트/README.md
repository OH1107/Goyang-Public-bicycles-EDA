# 코로나 이후 소비트렌드 분석 인사이트

## 개요
### 배경
- **소비트렌드 코리아2020**

### 목적
- 언택트, 뉴노멀 시대와 더불어 코로나 바이러스 발생 이후 급변하는 소비 행태를 분석 및 시각화를 실시
- 이를 바탕으로 2020년 코로나 이후 소비트렌드 제시 및 향후 소비 행태 인사이트 도출을 목표로 함
